### CLDC 1.1 Compatibility Rules

**âŒ DON'T USE (Java 5+ features):**
```java
// Generics
ArrayList<String> list;              // Use Vector
HashMap<K,V> map;                    // Use Hashtable

// Enhanced for loops
for (String s : list)                // Use classic for

// String methods
String.split(regex)                  // Parse manually
String.replace(String, String)       // Use substring()
String.isEmpty()                     // Use length() == 0

// Character methods
Character.isLetter(c)                // Use (c >= 'a' && c <= 'z')
Character.isDigit(c)                 // Use (c >= '0' && c <= '9')

// StringBuilder
StringBuilder sb;                    // Use StringBuffer

// Autoboxing
Integer.valueOf(5)                   // Use new Integer(5)
```

**âœ… DO USE (CLDC 1.1 compatible):**
```java
Vector list = new Vector();
Hashtable map = new Hashtable();
StringBuffer sb = new StringBuffer();

for (int i = 0; i < list.size(); i++) {
    Object item = list.elementAt(i);
}

// Manual string operations
String.substring(int, int)
String.indexOf(char)
String.charAt(int)
```

sb.append("mycommand    - Description\n");
